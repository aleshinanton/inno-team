4) Модель "Определение транспортных тегов текста".

Имя модели: gettags
URL модели //127.0.0.1:1937/predict/gettags

Статус модели: БОЕВАЯ
Описание модели: по входящему тексту возвращает наиболее релевантные транспортные теги
Данные для модели: ТЕХНОЛОГИЧЕСКИЕ НАПРАВЛЕНИЯ с сайта https://ftim.ru/ и НУЖНО ЕЩЕ качественное семантическое ядро
ЧТО ЕЩЕ НУЖНО для качества модели - это информация от экспертов по направленияю:
- определить семантическое ядро для каждого направления/поднаправления, то есть слова с наибольшими значением mutual information относительно поднаправления
- построить NER модель для устойчивых выражений для подкатегорий, например для фраз типа "нейронные сети", "беспилотный автомобиль" и т.д.
- посторить словарь синонимов для NER и т.п.
- настроить и проверить списки СТОП и НЕ_СТОП слов
- проверить качество ядра слов и выражений для каждого поднаправления (тут может быть оценка алгоритмом внутренняя и подтвердить эту оценку экспертной)

Входные данные:
    {
        "text": "Тут может быть текст из презентации стартапа или текст проблемы от экспертов или горожан или идеи от горожан. Проблема частых дтп автобусов"
    }

Предсказание модели:
    taglist = '#дтп #автобус'
